## 🧮 **2. MATHEMATICAL ANALYSIS**

### For Students/Children:
Parents can see advanced mathematical metrics calculated by the system:

#### Learning Velocity
```
Formula: (current_mastery - previous_mastery) / days_elapsed
```
- Shows rate of improvement
- Positive = getting better
- Negative = struggling
- Visible in Overview and Alerts tabs

#### Mastery Level Calculation
```
Formula: 0.4 × recent_score + 0.3 × avg_score + 0.3 × consistency
```
- Weighted combination of performance
- 0 to 1 scale (shown as percentage)
- Updates after each quiz
- Visible in Overview tab

#### Knowledge Decay Prediction
```
Formula: decay_date = last_attempt + (mastery_level × 30 days)
```
- Based on Ebbinghaus forgetting curve
- Higher mastery = longer retention
- Predicts when review needed
- Visible in Alerts tab

#### Engagement Score
```
Formula: (time_spent / expected_time) × (attempts / expected_attempts)
```
- Normalized 0-1 range
- Measures involvement
- Helps identify disengagement
- Visible in Alerts tab

### For Class Analysis:
Parents also get context about class performance:

#### Class Entropy (Uniformity)
```
Formula: entropy = -Σ(p_i × log2(p_i))
where p_i = proportion of students at mastery level i
```
- Shows knowledge distribution
- Low = everyone at similar level
- High = wide variation in understanding
- Teacher-level metric (indirect benefit to parents)

## 🔢 Mathematical Metrics - Parent-Friendly Explanation

### What Parents Need to Know:

#### Mastery Level (0-100%)
- **80-100%**: Excellent understanding
- **60-79%**: Good understanding, some practice needed
- **40-59%**: Needs more work
- **Below 40%**: Requires immediate attention

#### Learning Velocity
- **Positive**: Child is improving! Keep it up!
- **Near Zero**: Stable performance, maintaining level
- **Negative**: Struggling or losing knowledge, needs support

#### Knowledge Decay
- **Green (>14 days)**: No immediate action needed
- **Yellow (7-14 days)**: Plan a review session soon
- **Red (<7 days)**: Review urgently to prevent forgetting

#### Engagement Score
- **High (>70%)**: Child actively learning
- **Medium (40-70%)**: Moderate involvement
- **Low (<40%)**: May need motivation or different approach
